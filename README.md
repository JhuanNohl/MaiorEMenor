# MaiorMenor

Atividade acadêmica desenvolvida em Java.

## Objetivo

Construir uma classe chamada `MaiorMenor` que recebe um array nativo de inteiros e identifica o maior e o menor valor presente no array.

## Estrutura

A classe `MaiorMenor` possui os seguintes atributos:

- Um array de inteiros
- Uma variável para armazenar o maior valor
- Uma variável para armazenar o menor valor

## Funcionamento

Os elementos do array são gerados na classe `Main` e passados como parâmetro para a classe `MaiorMenor`.

O método `encontrarMaiorMenor()` percorre o array e determina qual é o maior e o menor valor inserido.

## Arquivos

- `MaiorMenor.java`
- `Main.java`
